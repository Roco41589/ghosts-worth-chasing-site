# DEPLOYMENT GUIDE - Ghosts Worth Chasing Website

## You Are Here

You have a complete, working Next.js website ready to deploy. Everything is built and configured.

## What You Need to Do (Simple Path)

### Step 1: Download This Folder
- You'll receive a ZIP file
- Download it to your computer
- Unzip it

### Step 2: Upload to GitHub
1. Go to: https://github.com/Roco41589/ghosts-worth-chasing-site
2. Click "uploading an existing file" (blue link on the page)
3. Drag ALL the files from the unzipped folder onto the GitHub page
4. Scroll down and click "Commit changes"

### Step 3: Deploy to Vercel
1. Go back to your Vercel dashboard (vercel.com/dashboard)
2. Click "Add New..." → "Project"
3. You should see "ghosts-worth-chasing-site" in your GitHub repositories
4. Click "Import" next to it
5. Leave all settings as default
6. Click "Deploy"
7. Wait 2-5 minutes

### Step 4: Connect Your Domain
1. After deployment succeeds, click "Settings" in Vercel
2. Click "Domains" in the left sidebar
3. Type: ghostsworthchasing.org
4. Click "Add"
5. Vercel will show you DNS records to add
6. Go to Squarespace domain settings
7. Add the DNS records Vercel shows you
8. Wait 10-30 minutes for DNS to propagate

### Step 5: You're Live!
- Your site will be at ghostsworthchasing.org
- It's fully functional
- No more work needed

## Troubleshooting

### "Build failed" in Vercel
- Tell Claude the error message
- Claude will fix and give you updated files

### Domain won't connect
- Double-check DNS records in Squarespace
- Make sure there are no typos
- Wait longer (DNS can take up to an hour)
- Try visiting in incognito mode

### Need to update content later
- Edit the files in GitHub directly (GitHub has a web editor)
- Or tell Claude what to change
- Vercel automatically redeploys when you update GitHub
